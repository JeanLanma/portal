# Portal

Portal para manejar los datos de la plataforma para clientes de la empresa.

## Stack

- [vue-3](https://vuejs.org/)
- [laravel-10](https://laravel.com/)

## Dependencias

- [inertia-vue3](https://inertiajs.com/)
- [tailwindcss-3](https://tailwindcss.com/)
- [laravel-jetstream](https://jetstream.laravel.com/introduction.html)
- [Laravel-Permissions](https://https://spatie.be/docs/laravel-permission/v5/introduction)
